# QR Chunker

QR Chunker is an experiment on streamlet, It uses streamlet to start an application that makes QR Images from Text and Files.

## Installation

### Docker

```
docker pull ahmedralmansoori/qrchunker
docker run -p 8501:8501 qrchunker
```
